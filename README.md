# UnityCreateFoldersInBulk

一个Unity批量创建文件夹的扩展编辑器

- 批量创建预制好的文件夹目录
- 可以自定义批量创建文件夹
- 支持导出/导出 .txt 配置文件

下载链接：[Download Unitypackage](https://github.com/MingMingZiX/UnityCreateFoldersInBulk/releases/download/0.9/CreaterFolder.unitypackage)
